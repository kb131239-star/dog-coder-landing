# 강아지 행동 분석 웹 서비스 랜딩 페이지 기획서

## 1. 프로젝트 개요
- **서비스명:** (가칭) 멍멍해석기 / Dog-Coder
- **목적:** 반려견의 특이한 행동이나 일상적인 행동의 의미를 AI(Gemini Flash)를 통해 빠르고 정확하게 분석하여 보호자에게 알려주는 서비스.
- **타겟 유저:** 강아지의 속마음이 궁금한 초보 및 모든 반려견 보호자.

## 2. 기술 스택
- **프레임워크:** Next.js (React 기반), Node.js 환경
- **배포:** Vercel (빠른 CI/CD 및 Next.js 최적화)
- **AI 모델:** Google Gemini Flash API (빠른 응답 속도와 멀티모달 기능 지원으로 텍스트 및 이미지 처리 적합)
- **스타일링:** Tailwind CSS (권장)

## 3. 핵심 기능 (MVP)
1. **행동 입력 (멀티모달 지원):**
   - 텍스트 입력: "강아지가 자꾸 자기 꼬리를 물려고 뱅글뱅글 돌아요"
   - 이미지/영상 업로드: 강아지의 행동이 담긴 사진이나 짧은 영상 업로드 (Gemini의 멀티모달 기능 활용).
2. **AI 분석 및 결과 제공:**
   - Gemini Flash 모델을 활용한 즉각적인 행동 의미 분석.
   - 보호자를 위한 대처 방법 및 팁 제공.
3. **공유 기능:**
   - 분석 결과를 이미지나 링크 형태로 SNS(인스타그램, 트위터 등)에 공유.

## 4. 랜딩 페이지 구조 (UI/UX)
- **Header:** 서비스 로고, '분석하기' 앵커 링크
- **Hero Section:** 
  - 캐치프레이즈: "우리 강아지, 지금 무슨 말을 하고 싶은 걸까요?"
  - 서브 텍스트: "텍스트나 사진을 올리면 AI가 강아지의 마음을 1초 만에 번역해 드립니다."
  - CTA 버튼: "지금 바로 행동 분석하기"
- **How it Works (이용 방법):**
  - Step 1: 행동 묘사 또는 사진 업로드
  - Step 2: Gemini AI의 초고속 분석
  - Step 3: 우리 아이 마음 확인 및 솔루션 받기
- **Demo / Main Action Section:**
  - 사용자가 실제로 텍스트를 입력하거나 파일을 업로드할 수 있는 폼(Form).
  - 결과 출력 영역 (스켈레톤 로딩 애니메이션 포함).
- **Use Cases (예시 사례):**
  - "보호자를 핥는 이유", "바닥을 파는 행동" 등 흥미를 유발하는 샘플 결과 카드 제공.
- **Footer:** 이용약관, 개인정보처리방침, 문의처

## 5. 아키텍처 및 보안 (Next.js & Vercel)
- **Frontend & Backend 통합:** Next.js의 Route Handlers (API Routes)를 사용하여 프론트엔드와 서버리스 백엔드를 한 번에 구성.
- **API 보안:** Gemini API Key는 클라이언트 사이드에 절대 노출되지 않도록, Vercel의 환경 변수(Environment Variables)로 등록하고 Next.js API Route(Node.js 환경)에서만 호출하여 사용.
- **배포 파이프라인:** GitHub 리포지토리를 Vercel에 연결하여 코드 푸시 시 자동 배포(CI/CD)되도록 설정.

## 6. 향후 확장 계획
- 견종 및 나이별 맞춤형 분석 고도화
- 사용자 로그인 기반 강아지 프로필 저장 (다견 가정 지원)
- 이전 분석 기록 히스토리 확인 기능
